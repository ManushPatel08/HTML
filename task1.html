C Language

What is C?

- C is a general-purpose programming language created by Dennis Ritchie at the Bell Laboratories in 1971 1972.
- It is a very popular language, despite being old. The main reason for its popularity is because it is a fundamental language in the field of computer science.


Advantages:

-> C is a fast and reliable language that can write efficient code.
-> C is extendable and has built-in functions and operators.
-> C is more efficient than C++ because it doesn't require Virtual Method Table (VMT) lookups.

Disadvantages:

-> C does not support exception handling.
-> It does not have strict type checking.
-> Most importantly, C does not support OOPs.
